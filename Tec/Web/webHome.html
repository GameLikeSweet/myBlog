        Web개발이란?

        웹 개발은 웹사이트와 웹 애플리케이션을 구축하고 유지보수하는 과정을 말합니다. 크게 프론트엔드 개발, 백엔드 개발, 풀 스택 개발의 세 가지 영역으로 나눌 수 있습니다.

        웹 개발에 사용되는 주요 기술들

        HTML (Hypertext Markup Language) :

        웹 브라우저에 표시되도록 설계된 문서의 표준 마크업 언어이다. HTML은 웹 콘텐츠 의 내용과 구조를 정의한다.

        CSS (Cascading Style Sheets) :

        HTML 이나 XML과 같은 마크업 언어로 작성된 문서의 표현과 스타일을 지정하는데 사용되는 스타일 시트 언어이다.

        JavaScript :

        동적으로 변경되는 콘텐츠를 만들고, 멀티미디어를 제어하고, 이미지에 애니메이션을 적용하는 등 거의 모든 작업을 수행할 수 있는 스크립팅 언어이다.
