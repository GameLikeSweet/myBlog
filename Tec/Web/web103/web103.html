        Position

        엘리먼트에 position을 적용하여 해당 엘리먼트의 위치를 조정한다.

        1. static : 기본값으로 일반적인 문서 배치 기준을 따른다.
        2. relative : 일반적인 문서 배치 기준에 따라 배치하고 자기 자신을 기준으로 top, right, bottom, left 값에 따라 오프셋을 적용한다.
        3. absolute : 일반적인 문서 배치가 아니라 가장 가까운 지정 조상 엘리먼트를 대상으로 배치한다.
        4. fixed : 일반적인 문서 배치 기준이 아니라 뷰포트(전체 화면)을 기준으로 배치된다. 스크롤에 영향을 받지 않고 화면 특정 지점에 고정된다.

        모든 박스를 대상으로 top: 10%를 주었을 때 static 박스는 top의 영향을 받지 않는게 보인다. 반면 나머지 positoin들은 모두 top 10%의 영향을 받는다.

        relative 박스는 static 박스 내부의 h2텍스트의 영향을 받아 해당 텍스트를 기준으로 위에서 10%를 내려갔다.

        absolute 박스는 static 박스와 다르게 relative 박스 내부에 h2텍스트 영향을 받지 않고 relative 박스를 기준의로 위에서 10%를 내려가서 relative 박스의 텍스트와 약간 겹치게 있다.

        fixed의 경우 클릭을 통해 position:fixed를 활성화 하면 스크롤을 통한 화면 이동과 무관하게 화면 전체 기준으로 위에서 10%에 고정되어 있다.

        style을 css 파일로 분리하여 index.html 내부에서 style 엘리먼트를 찾을 수 없다. 대신 link 엘리먼트를 이용하여서 외부에 있는 css를 불러와 적용시키는 것을 볼 수 있다. hader, nav등의 식별자로 엘리먼트들을 나누어 가시성과 관리의 편의성을 높혔다.
