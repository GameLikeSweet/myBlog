        GitHub 란?

        GitHub는 코드를 저장하고 공유하고 다른 사람과 함께 코드를 작성할 수 있는 클라우드 기반 플랫폼이다.

        GitHub의 기본적인 기능 중 하나인 공동 작업은 GitHub가 빌드된 오픈 소스 소프트웨어인 Git으로 지원된다.

        Git 과 GitHub

        GitHub에 파일을 업로드하면 파일을 “Git 리포지토리”에 저장합니다. 즉, GitHub에서 파일을 변경하거나 “커밋”하면 Git에서 변경 내용을 자동으로 추적하고 관리하기 시작한다.


        Git clone

        git clone이란 다른 프로젝트에 참여하려거나(Contribute) Git 저장소를 복사하고 싶을 때 git clone 명령을 사용한다.

        git clone을 통해 프로젝트를 복사하면 프로젝트 히스토리를 전부 받아온다.

        git clone https://github.com/GameLikeSweet/myBlog

        해당 명령을 실행하면 새 디렉토리를 만들고 그 안에 url에 있는 remote 저장소의 데이터를 가져와 가장 최신 버전으로 세팅한다.

        Git push

        git push는 git에 설정한 remote 저장소에 현제 git으로 관리하고 있는 데이터를 보내는 명령이다.

        git push origin main

        해당 명령어를 통해서 git에 origin이라 이름으로 저장된 remote 저장소에 main 브런치에 현제 작업 데이터를 보낼 수 있다.
